# Screening Criteria / 優良企業スクリーニング基準

Stockpile を使って優良企業を選定する際の基準と手法を定義する。

---

## 1. 基本方針

「優良企業」の定義は投資目的によって異なる。本ドキュメントでは以下の 4 軸で総合評価する:

| 軸 | 重視する指標 | 重み |
|---|---|---|
| **収益性** | 営業利益率, 純利益率 | 高 |
| **資本効率** | ROE, ROA | 高 |
| **財務健全性** | 自己資本比率, D/E | 中 |
| **成長性** | 売上成長率, 純利益成長率 | 中 |

---

## 2. スクリーニング基準

### Tier 1: 最低条件 (足切り)

これを満たさない銘柄は対象外とする:

| 指標 | 条件 |
|---|---|
| 営業利益率 | > 5% |
| ROE | > 8% |
| 自己資本比率 | > 20% (金融除く) |
| 売上成長率 | > -5% (縮小企業を除外) |

### Tier 2: 優良企業の条件

| 指標 | 条件 | 根拠 |
|---|---|---|
| 営業利益率 | > 15% | 業種平均を大きく上回る |
| ROE | > 15% | 株主資本を効率的に活用 |
| 自己資本比率 | > 40% | 景気後退に耐えうる |
| D/E | < 0.5 | 過剰な借入依存でない |
| 売上成長率 | > 5% | 持続的な成長 |

### Tier 3: 卓越した企業の条件

| 指標 | 条件 | 該当例 |
|---|---|---|
| 営業利益率 | > 30% | キーエンス, 中外製薬, MSFT |
| ROE | > 25% | リクルート, 東京エレクトロン |
| 純利益成長率 | > 20% | 東京エレクトロン, GOOGL |
| D/E | < 0.1 | キーエンス, 信越化学, NVDA |

---

## 3. スクリーニング手順

### Step 1: 候補銘柄のリストアップ

業種代表、時価総額上位、テーマ（AI, EV等）から 10-20 銘柄を選定。

### Step 2: データ取得

```bash
# 一括比較 (最大5銘柄ずつ)
stockpile metrics compare 6861 4063 4519 8035 6098 --market jp --format json

# 個別に詳細確認
stockpile metrics get 6861 --market jp
```

### Step 3: Tier 1 足切り

営業利益率 > 5%, ROE > 8% を満たさない銘柄を除外。

### Step 4: Tier 2/3 で評価

4 軸ごとにランキングし、総合スコアをつける。

### Step 5: 定性分析の補完

```bash
# ニュースで最新動向を確認
stockpile news get 6861 --market jp

# 株価推移で市場評価を確認
stockpile price get 6861 --market jp --start 2024-04-01
```

## 4. 評価マトリクス テンプレート

分析レポートでは以下の形式で評価を記載する:

```
| 銘柄 | 営業利益率 | ROE | 自己資本比率 | D/E | 売上成長 | 純利益成長 | 総合 |
|---|---:|---:|---:|---:|---:|---:|---|
| XXXX | XX.X% | XX.X% | XX.X% | X.XX | +X.X% | +X.X% | S/A/B/C |
```

### 総合評価基準

| ランク | 条件 |
|---|---|
| **S** | Tier 3 を 3 項目以上クリア。弱点がほぼない |
| **A** | Tier 2 を全項目クリア。Tier 3 を 1-2 項目クリア |
| **B** | Tier 2 を概ねクリア。一部に課題あり |
| **C** | Tier 1 はクリアするが Tier 2 に届かない項目がある |

---

## 5. 注意事項

- **過去データは将来を保証しない。** 財務指標はあくまで過去の実績
- **業種特性を考慮する。** 金融・商社・小売は利益率が構造的に低い
- **1期だけで判断しない。** `stockpile metrics get` で 3-4 期分の推移を確認する
- **定量 + 定性。** 数字だけでなくニュース・事業構造も確認する
- **本ドキュメントは投資助言ではない。** 投資判断は自己責任で行うこと
